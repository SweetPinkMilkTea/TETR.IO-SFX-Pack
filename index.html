How to install this pack

To install this pack, follow these steps:

Ensure TETR.IO PLUS is installed

Make sure you have TETR.IO PLUS installed. If you haven't, download it...

...and install it inside your TETR.IO installation folder by replacing the existing app.asar with the downloaded one.

On windows, the TETR.IO installation folder is usually located at
C:\Users\{user}\AppData\Local\Programs\tetrio-desktop\resources.

Applying the SFX Pack

After downloading the SFX pack, extract the contents of the zip somewhere accessible.

Under the Sound Effects section in the TETR.IO PLUS menu, enable custom SFX and enter the SFX Editor.

Click "Choose Files" under "Replace multiple by filename" and select all extracted files.

Save everything by clicking "Re-encode and save changes". This process takes a moment, during which the menu becomes unresponsive. Don't close the window during that time.

After the process is complete, you can close all settings windows and restart TETR.IO by executing a hard refresh (CTRL + F5).
